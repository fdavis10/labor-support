# Админка новостей (кастомная)

- **Публичный API новостей:** `GET /api/news/` — список опубликованных новостей для главной страницы.
- **Админ-API** доступен только по скрытому пути (по умолчанию `c8k2m9p5`). URL нельзя угадать и он не ссылается с сайта.

## Как попасть в админку

1. Создайте суперпользователя (если ещё нет):
   ```bash
   python manage.py createsuperuser
   ```
2. Запустите бэкенд и фронт. В браузере откройте:
   ```
   http://localhost:5173/c8k2m9p5
   ```
   (или ваш домен + путь; путь задаётся в `ADMIN_API_PREFIX` на бэкенде и `VITE_ADMIN_PATH` на фронте.)
3. Введите логин и пароль суперпользователя. Токен создаётся при первом входе.
4. В админке можно добавлять, редактировать и удалять новости. Поле «Опубликовано» управляет показом на сайте.

## Переменные окружения

- **Бэкенд:** `ADMIN_API_PREFIX` — префикс для админ-API (по умолчанию `c8k2m9p5`). Меняйте в production на свой секретный путь.
- **Фронт:** `VITE_ADMIN_PATH` — тот же путь (по умолчанию `c8k2m9p5`).

## Миграции

```bash
python manage.py migrate
```

После этого доступны модель `News` и эндпоинты.
